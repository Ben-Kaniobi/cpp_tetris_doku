\chapter{Fortsetzung}\label{ch:fortsetzung}
	%
	Eine Weiterführung des Projektes ist nicht vorgesehen. Trotzdem lassen sich Use-Cases kreieren, in denen eine Weiterführung des Projekts wünschenswert wäre. Als mögliche Erweiterung könnten beispielsweise folgende Funktionen umgesetzt werden:
	%
	\begin{itemize}
		\item{\textbf{Spiel pausieren}: Eine Funktion zum vorübergehenden Unterbrechen eines laufende Spiels könnte umgesetzt werden}
		%
		\item{\textbf{Punktestand festhalten}: Für das laufende Spiel könnten Punkte gesammelt werden. Diese Funktion wäre insbesondere in Kombination mit einer Bestenliste vorstellbar}
		%
		\item{\textbf{Mehrspielermodus}: Mit grossem Aufwand könnte das Spiel für einen Mehrspielerbetrieb umgerüstet werden} 
	\end{itemize}
